== Description ==

Have a crap ton of sidebars and widgets setup?

Losing sleep because you're afraid to lose all those settings if someone messed with it? 

Look no further. Use this widget to 

1. Take snapshots of your current widget settings.
2. Restore previous snapshots.
3. Import / Export snapshots as XML.


== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new set of Widget Snapshot commands will appear on the top of the Widgets page.


== Frequently Asked Questions ==

= Why can't I restore my sidebar? =

This plugin only restores widget settings of sidebars, not the sidebars themselves.

= Some widgets didn't get restored. Y U NO RESTORE!? =

Widgets will only be restored if their original sidebars are still present. This plugin only restores widget settings of sidebars, not the sidebars themselves.
